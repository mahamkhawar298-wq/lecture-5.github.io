# WILD ZINE | Avant-Garde Nature Fashion

A digital fashion magazine platform celebrating the intersection of nature and haute couture.

## Features

- Fullscreen welcome overlay with entrance animation
- Dynamic search filtering across all content
- 4 high-resolution editorial fashion images
- Responsive 12-column CSS Grid layout
- Interactive quote generator
- Contact section with publication details

## Tech Stack

HTML · CSS (Grid, Flexbox, Gradients)

## Quick Start

Open `index.html` in any modern browser.
